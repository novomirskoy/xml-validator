# Xml validator

Валидация xml документов с помощью xsd схем

## Как пользоваться

Необходимо создать экземпляр класса `Validator` и вызвать метод `validate()`
передав в него xml документ в виде строки и схему. Схема может быть как **строкой**, так и **путём к xsd 
файлу**.
Создать экземпляр схемы можно с помощью именованных конструкторов `Schema::file()`
или `Schema::string()`, передавая туда путь к файлу или строку со схемой соответственно.

Валидатор не хранит результат валидации, поэтому сразу возвращает объект типа Result. Который скажет
о том была ли валидация успешной, а так же вернёт список всех возникших ошибок.

## Как пользоваться

```
// Если xsd схема в виде строки
$result = new Validator()->validate(xml: $xml, schema: Schema::string(''))

// Если xsd схема в виде файла
$result = new Validator()->validate(xml: $xml, schema: Schema::file('/tmp/schema.xsd'))

if (!$result->isValid()) {
    var_dump($result->errors);
}
```
